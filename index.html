Projects

Nonogram Solver

A website I developed to solve nonograms, a type of number grid puzzle that creates images. It uses Admin-LTE for the UI, and solves nonograms based on a recursive algorithm that searchs through all valid row and column permutations.

Data Ocean

This is a subsection of a site I worked on to display and query submarine data, as well as manage users and view AWS instances/buckets. The full version of the site uses a Django backend with a Postgres database. This subsection just uses some JSON structures to display fake data.

Subnautica Deep Dive

This was created as part of my CSC492 class, Human Computer Interaction. It is an informational site for a game called Subnautica, and uses Bootstrap and various JavScript libraries to display information in a (hopefully) visually interesting way.

Subnautica Deep Dive Process

This is the creation process for the Subnautica Deep Dive site for CSC492. This process was done in a group and demonstrates knowledge of the web development process, including concepts like problem statements, target users, use cases, storyboards, wireframes, a Figma Prototype, and user testing.

Mandala Maker

Part of an example site called "The Sandbox" made for CSC372: Dynamic Web Design and Programming. Mandala Maker is an adaptation of a Khan Academy project I did, expanded to a site using the canvas API. It also includes an API for exporting mandalas as images.

Overcooked

An old site I made for CSC271: Web Design and Programming. Although not dynamic, it demonstrates some core web design skills without using any external libraries. It also inlcudes links to my CSC106 website, which houses my original Manadala Maker.
